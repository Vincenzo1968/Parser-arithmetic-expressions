C Version:

To compile with GCC:

Linux:

    gcc -Wall -W -std=c99 -O3 -lm ParseExpr.c parser.c lexer.c -o exprparser

Windows:

    gcc -Wall -W -std=c99 -O3 -lm ParseExpr.c parser.c lexer.c -o exprparser

To run the program:

Linux:

    ./exprparser

Windows:

    exprparser

To exit the program, press ENTER without entering anything.
